System Requirements
====================

1. `KBase Workspace <https://github.com/kbase/workspace_deluxe>`_ (See section on `Deploying the Workspace Service locally <https://github.com/kbase/workspace_deluxe/blob/dev-candidate/docsource/developers.rst>`_)

2. `ElasticSearch v5.5.2 <https://www.elastic.co/downloads/past-releases/elasticsearch-5-5-2>`_. Note: the current implementation is not compatible with ElasticSearch v6+.

3. `Kibana v5.5.2 <https://www.elastic.co/downloads/past-releases/kibana-5-5-2>`_
